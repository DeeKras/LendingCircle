# LendingCircle

This app helps you keep track of things you lent - to whom, when, when to expect it back, what condition it was in.

And you can mark when it was returned and in what condition.

--
Created in django and python.

----

Phase 2 will allow the user to rate the borrower.

--

Later phase might keep track about borrowers - how soon they return things and in what condition.
